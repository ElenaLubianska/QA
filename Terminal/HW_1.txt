3) Зайти в папку = cd (name folder)
4) Создать 3 папки = mkdir {name folder,name folder,name folder}
5) Зайти в любоую папку = cd (name folder)
6) Создать 5 файлов (3 txt, 2 json) = touch {name_file.txt,name_file.txt,name_file.txt,name_file.json,name_file.json}
7) Создать 3 папки = mkdir {name folder,name folder,name folder}
8. Вывести список содержимого папки = ls -l
9) + Открыть любой txt файл = vim (name file).txt
10) + написать туда что-нибудь, любой текст. = i (write something)
11) + сохранить и выйти. = esc -> :wq -> enter
12) Выйти из папки на уровень выше = cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку. = mv name folder/{name_file,name_file} name folder

15) Найти файл по имени = find -name "name_file"

17) вывести несколько первых строк из текстового файла = head -n2 name_file.txt
18) вывести несколько последних строк из текстового файла = tail -n2 name_file.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. less 1.txt
